BMW Stock Locator API

https://stolo-data-service.prod.stolo.eu-central-1.aws.bmw.cloud

Endpoints

POST /vehiclesearch/aggregate/{locale}/stocklocator?brand={brand} ▶
Get total count and aggregated filter values (series, drive types, models) for available vehicles.

Path Parameters

locale: e.g., de-ch, fr-ch, it-ch

Query Parameters

brand: BMW

Request Body

{ "searchContext": [{}], "resultsContext": { "sort": [ { "by": "PRICE", "order": "ASC" } ] } }

Response
POST /vehiclesearch/search/{locale}/stocklocator?maxResults={n}&startIndex={i}&brand={brand}&context=results-page ▶
Search vehicles with pagination. Returns detailed vehicle information including pricing, specifications, and media.

Path Parameters

locale: e.g., de-ch, fr-ch, it-ch

Query Parameters

brand: BMW
maxResults: 1-12 (max 12 per page)
startIndex: pagination offset (default: 0)
context: results-page

Request Body

{ "searchContext": [{}], "resultsContext": { "sort": [ { "by": "PRICE", "order": "ASC" } ] } }
